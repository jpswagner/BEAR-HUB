# BEAR-HUB ↔ Bactopia 4.0 — Real-data validation

Validation of the Reflex BEAR-HUB command generation against real sequencing data,
run through the actual Bactopia 4.0 pipeline via Docker.

## Test data
- **Short reads (Illumina PE):** `Streptococcus pyogenes` (pyogenes-10, 151bp, ~9MB/read)
- **Long reads (ONT):** `Enterobacter` (genomas_enterobacter GECL####, 1.7–12kb reads, 304MB–1.5GB)
- **Assembly (FASTA):** ref_strains/BEN2908.fa

## Bugs found ONLY by running real data (unit tests could not catch these)

| # | Bug | Symptom | Fix |
|---|---|---|---|
| 1 | `--ident_min`/`--coverage_min` (AMRFinder) + `--scheme`/`--minscore`/`--nopath` (MLST) emitted | `Parameter ident_min is not declared` — Nextflow aborts | These are Bactopia **Tools** params, not main-pipeline. Removed; tools run with defaults. |
| 2 | FOFN had a single `extra` column | `Missing required column(s) 'se', 'ont', 'assembly'` | Rewrote `build_fofn` to the canonical 9-column header `sample runtype genome_size species r1 r2 se ont assembly` (matches `bactopia prepare`). |
| 3 | `_collect()` used `Path.resolve()` (follows symlinks) | symlinked data mis-classified (ONT→single-end, hybrid not merged) | Switched to `os.path.abspath` (no symlink follow). |
| 4 | `-profile` hardcoded to docker | UI profile selector ignored | Threaded `self.profile` into `_main_cmd`. |
| 5 | `--max_memory "16 GB"` (space → quoted) | malformed value | `16.GB` dotted notation. |

> **Lesson:** the param audit JSON summed *all* module params (incl. bactopia-tools),
> so Tools-only params looked valid. Only a real run distinguishes main-pipeline from
> Tools params, and only real data exercises the FOFN format + symlink handling.

## End-to-end result — short reads ✅

Command generated by BEAR-HUB (PE Unicycler, 4 cpus, 8GB, docker) ran the **full
Bactopia 4.0 pipeline successfully**:

```
[SUCCESS] completed=11 failed=0 cached=0   Duration: 3m 51s   Success: true
```

Outputs produced and verified:
- **Assembly:** 67 contigs, **1,783,095 bp** (expected ~1.8 Mb for S. pyogenes), N50 38,595
- **MLST:** `spyogenes ST 28, PERFECT, score 100`
- **AMRFinderPlus:** ran with internal defaults (confirms removing top-level AMR params was correct)
- **Annotation (Prokka), Sketcher (mash/sourmash):** completed
